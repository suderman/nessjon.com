nessjon.com
===========

Wedding Website for Jonathan Suderman & Janessa Sheppard  
<http://nessjon.com>

Installation
------------

1. `brew install node`
2. `npm install -g grunt-cli bower foreman`
3. `git clone git@github.com:suderman/nessjon.com.git && cd nessjon.com`
4. `npm install`
5. `bower install`
6. `grunt`
7. `foreman start`
